## ✨ Overview

Prompt engineering for Claude requires a different approach than other LLMs due to its reliance on XML tags and long-context capabilities. **Awesome Claude Prompts** provides highly specific, optimized prompts designed to get maximum performance out of the Claude 3 and 3.5 model families.

Whether you're building an autonomous coding agent, a creative writing assistant, or a strict JSON data extractor, you'll find a proven template here.

## 🗂️ Categories

*   **👨‍💻 Software Engineering**: Refactoring, code reviews, test generation, and architecture design.
*   **✍️ Copywriting & SEO**: High-converting landing page copy, semantic SEO blog posts, and cold emails.
*   **📊 Data Analysis**: SQL generation, pandas manipulation, and unstructured data extraction.
*   **🛡️ Red Teaming & Security**: Security audits, vulnerability scanning, and jailbreak simulation.
*   **🎭 Personas**: Highly detailed system prompts for specialized virtual domain experts.

## 🚀 How to Use

1. Browse to your desired category directory.
2. Copy the system prompt.
3. Replace the `[BRACKETED]` placeholder variables with your specific context.
4. Enjoy improved outputs!
